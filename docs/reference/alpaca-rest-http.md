# alpaca-rest-http

Shared HTTP transport crate.

Key exports:

- `HttpClient`
- `HttpClientBuilder`
- `RetryConfig`
- `RetryDecision`
- `TransportObserver`
- `RequestParts`
- `ResponseMeta`
- `ErrorMeta`

docs.rs:

- <https://docs.rs/alpaca-rest-http>

This crate intentionally does not expose Alpaca resource models or endpoint-specific clients.
