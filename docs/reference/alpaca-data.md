# alpaca-data

Async market data client crate.

Resource accessors:

- `stocks()`
- `options()`
- `news()`
- `corporate_actions()`

Environment variables:

- `ALPACA_DATA_API_KEY`
- `ALPACA_DATA_SECRET_KEY`
- `ALPACA_DATA_BASE_URL`

- [Market Data API Coverage](../api-coverage/market-data.md)
